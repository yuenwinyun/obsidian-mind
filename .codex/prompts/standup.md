# Standup

Run a morning kickoff for this vault.

Workflow:

1. Read `Home.md`
2. Read `brain/North Star.md`
3. Check `work/Index.md` and `work/active/`
4. Read today's and yesterday's daily note if Obsidian daily notes exist
5. List open tasks with Obsidian CLI if available
6. Review recent git activity from the last 24 hours
7. Check for obvious inbox or orphan-note cleanup work

Present a concise standup with:

- Yesterday: what changed
- Active Work: current projects and status
- Open Tasks: highest-signal pending work
- North Star Alignment: how current work maps to goals
- Suggested Focus: what to prioritize next

Prefer QMD and Obsidian CLI when available. Fall back to direct file reads and `rg` when they are not.
