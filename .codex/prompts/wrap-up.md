# Wrap Up

Run an end-of-session review of the vault changes from this session.

Workflow:

1. List notes created or modified
2. Verify note hygiene:
   - frontmatter present
   - correct folder
   - at least one `[[wikilink]]`
3. Check index consistency:
   - `work/Index.md`
   - `brain/Memories.md`
   - `org/People & Context.md`
   - `perf/Brag Doc.md`
4. Check for orphan notes or stale active-project status
5. Look for wins, patterns, or gotchas that should be recorded
6. Suggest small workflow improvements if this session revealed friction

Present the result as:

- Done
- Fixed
- Flagged
- Suggested

Prefer fixing small structural issues directly. Flag larger or ambiguous changes for the user.
